# Turtle Race

A small Turtle graphics project made during Angela Yu's Python course.

The player can bet on a turtle color, then watch the race to see which turtle wins.

---

## Features

- Multiple turtle racers
- Random turtle movement
- Betting system
- Winner detection
- Turtle graphics animation

---

## What I Practiced

- Lists
- Loops
- Object creation
- Storing objects in lists
- Random movement
- Basic game loop logic

---

## How to Run

Run the project with:

```bash
python main.py
```
